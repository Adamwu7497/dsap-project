# 台北市救護車派遣模擬與分析

## Proposal Report

### 動機與目標
當意外發生時，及時將病患送往醫院往往是很重要的。
因此，我想透過分析2022~23的救護車派遣事件，
撰寫程式模擬台北市個區域的派遣事件，並分析台北市救護車的分布狀況提出建議。

### 預期功能
1.利用python將資料清洗成程式碼易讀的csv檔。
2.利用C++建立模擬系統，動態模擬派遣案件並回傳抵達時間。
3.分析回傳資料尋找台北市內需求量較大的區域。

### 使用技術
C++：使用為核心模擬系統撰寫的語言。
Python：使用為資料處理、資料分析的工具，並使用其進行結果視覺化。

### 時程規劃
第一週：清洗資料，輸出成格式統一的csv。
第二週：建立Station、ambulance、case等class和函式，存入各.h檔。
第三週：建立地圖系統，回傳時間。
第四週：幫每個station撰寫車輛管理系統。
第五週：寫出main迴圈。
第六週：debug和使用真實資料跑。
第七週：將結果視覺化，生出結論。
第八週：撰寫報告。
